# Frame Player Wiki

Frame Player is a frame-first video review tool for people who need to know exactly which frame they are looking at. It supports single-video review, two-pane compare review, frame stepping, frame jumps, loop review, and export workflows.

The Windows release remains the stable WPF app. Unified Avalonia Preview `0.2.0` is the current synchronized Windows/macOS preview. The earlier split preview releases are superseded and kept only for historical validation.
