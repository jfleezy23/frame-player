# Keyboard Shortcuts

| Action | Windows stable | macOS Preview | Windows Avalonia Preview |
| --- | --- | --- | --- |
| New window | `Ctrl+N` | `Cmd+N` | `Ctrl+N` |
| Open video | `Ctrl+O` | `Cmd+O` | `Ctrl+O` |
| Close video | `Ctrl+W` | `Cmd+W` | `Ctrl+W` |
| Export diagnostics | `Ctrl+Shift+E` | `Cmd+Shift+E` | `Ctrl+Shift+E` |
| Help | `F1` | Help menu | `F1` |
| Play or pause | `Space` | `Space` | `Space` |
| Previous frame while paused | `Left` | `Left` | `Left` |
| Next frame while paused | `Right` | `Right` | `Right` |
| Rewind 5 seconds | `,` | `,` | `,` |
| Fast forward 5 seconds | `.` | `.` | `.` |
| Set loop in | `[` | `[` | `[` |
| Set loop out | `]` | `]` | `]` |
| Toggle loop playback | `L` | `L` | `L` |
| Full screen | `F11` or `Alt+Enter` | native full-screen control | `F11` or `Alt+Enter` |
| Commit frame jump | `Enter` | `Enter` | `Enter` |

After a frame-number jump, focus returns to the video controls so immediate arrow-key stepping works without an extra click.
